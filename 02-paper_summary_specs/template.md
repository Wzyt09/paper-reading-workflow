# 自定义规范模板

把这份文件复制一份并改名，例如：

- `compact.md`
- `formula_first.md`
- `compare_two_papers.md`

之后你就可以直接说：

```text
按规范 compact 总结并打包 01-paper/xxx.pdf
```

我会默认去找：

```text
02-paper_summary_specs/compact.md
```
