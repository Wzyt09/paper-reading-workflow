## 对比主题

Rydberg 中性原子量子计算中的串扰、门误差和规模化控制。

## 综合结论

三篇文章都讨论 Rydberg 阵列中的非理想误差，但侧重点不同：论文 A 更像误差表征框架，论文 B 强调控制脉冲优化，论文 C 讨论规模化系统中的工程约束。如果目标是设计实验，应优先读 A 和 B；如果目标是判断路线可扩展性，C 更关键。

## 互相支持的结论

1. 三篇文章都说明 Rydberg 门误差需要拆分分析，不能只看最终保真度。
2. 论文 A 和 B 都支持“校准和控制优化可以显著降低特定误差”的判断。
3. 论文 C 提醒这种优化在规模化后可能受到并行操作、光场不均匀性和读出链路限制。

## 存在张力的地方

论文 B 对控制优化的效果比较乐观，但论文 C 表明当阵列规模增加时，控制复杂度和误差预算会显著上升。因此，B 的结论更适合小到中等规模阵列，不能直接当作大规模容错系统的证据。

## 下一步阅读建议

- 如果要复现实验：先精读论文 A 的测量流程，再读论文 B 的控制参数。
- 如果要写综述：把论文 C 作为架构层面的限制条件。
- 如果要做建模：优先提取三篇文章中的误差预算、阵列尺寸和门时间。
